Mastermind

Visit site open_in_new Source code open_in_new

Mastermind is a code-guessing game where you have to guess a colour code by guessing and using the clues from the guesses. Note: This game can only be played on a PC.

How to play
  • A 4-colour code is generated and you have 12 guesses to guess it. Each colour in the code is unique.
  • The number in the left white slider indicates the number of colours that you guessed correctly but in the wrong position.
  • The number in the right red slider indicates the number of colours that you guess correctly and in the correct position.
  • If a slider did not move, this means '0'.
  • To guess the code, click on the dots and press "Enter".
  • Do not have two of the same colour in your guess.

Connect 4

download Zip file Source code open_in_new

Connect 4 is a 2-player game where both players take turns placing their colour on a board. A player wins when a row/column/diagonal of 4 of that player's colour lines up.

How to play
  • Hover over any column and click the mouse. That player's colour will be dropped in that column if there is at least one empty space in that column.
  • Both players take turns dropping their colour until one player wins or all spaces have been filled, in that case it is a draw.

Space Invaders

download Zip file Source code open_in_new

In this game, your task is to shoot enemies that are coming down the screen while avoiding the enemies' projectiles.

How to play
  • You have a certain health; every time you get hit by an enemy's projectile, your health decreases.
  • You also have 3 lives. You lose a life when an enemy reaches the bottom of the screen.
  • You lose the game when all 3 lives are lost, or when your health decreases to 0.
  • There are different levels, and at each level, there will be a wave of enemies coming down the screen.
  • When all enemies are shot, you move on to the next level.
